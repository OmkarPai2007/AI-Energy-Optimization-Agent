# AI Energy Optimization Agent

## AICTE Generative AI & Agentic System Engineering Internship

### Overview

The AI Energy Optimization Agent is an AI-powered smart home assistant that analyzes household electricity consumption using Google Gemini AI.

It identifies excessive energy usage, provides personalized recommendations, supports SDG 13 (Climate Action), and generates professional energy optimization reports.

## Features

- Smart Home Energy Dataset
- Energy Usage Visualization
- AI Analysis using Google Gemini
- High Energy Consumption Detection
- SDG 13 Sustainability Recommendations
- AI-generated Report

## Technologies Used

- Python
- Google Colab
- Google GenAI SDK (`google-genai`)
- Pandas
- Matplotlib
- Gemini API

## SDG Supported

**SDG 13 – Climate Action**
